# vue3Ts
vue3源码解析与TS运用教学

## 项目初始化步骤

1. 安装依赖（首次使用需执行）：
	```bash
	npm install
	```

## 一键运行 TypeScript 示例文件

在项目根目录下，使用以下命令可直接编译并运行对应的 TS 示例文件：

```bash
# 运行 basic-types.ts 示例
npm run run:basic

# 运行 interface-demo.ts 示例
npm run run:interface

# 运行 function-demo.ts 示例
npm run run:function

# 运行 class-demo.ts 示例
npm run run:class
```

每条命令会自动编译并输出对应示例文件的运行结果。
